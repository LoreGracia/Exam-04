## Installation

To compile the program and generate the execution file, follow these steps:

### Clone this repository:

```bash
git clone [repository link]
cd [new folder]
```

### Compile the program:
```bash
cc micro.c
```

### Execution
Once you have compiled the files, you can execute it as follows:

```
./a.out [arguments]
```
